# Branch
## 독립적인 작업흐름을 만들고 관리한다
## Branch 명령어
>### 브랜치 생성
> git branch {branch name}
>### 브랜치 이동
> git chechout {branch name}
>### 브랜치 생성 및 이동
> git chechout -b {branch name}
>### 브랜치 목록
> git branch
>### 브랜치 삭제
> git branch -b {branch name}
## Merge
## branch에서 작업 후 각 이력을 합치기 위해 merge 명령어 사용
>### feature-a branch 이동 후 commit
>### master branch로 병합
